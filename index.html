¿Qué es Git?

Git es un sistema de control de versiones distribuido que permite registrar cambios en archivos y coordinar el trabajo entre varias personas en un proyecto de software.

Control de versiones

Permite guardar cada cambio realizado en el código.

Historial

Puedes volver a cualquier versión anterior.

Trabajo colaborativo

Muchos desarrolladores pueden trabajar en paralelo.

Branches

Permite crear ramas para desarrollar nuevas funcionalidades.

Configuración inicial de Git

Configurar nombre

git config --global user.name "Tu Nombre"

Configurar email

git config --global user.email "correo@email.com"

Ver configuración

git config --list

Estos comandos permiten que Git identifique quién realiza los cambios.

Flujo de trabajo en Git

1. Crear repositorio

git init

2. Agregar archivos

git add .

3. Crear commit

git commit -m "primer commit"

4. Conectar repositorio remoto

git remote add origin URL

5. Subir cambios

git push -u origin main
